## Operators

1. Arithmetic: + - * / % 
2. Assignment: = += -= *= /= %=
3. Comparision: > < >= <= != == ===
4. Logical: && || !
5. Bitwise: & | ^ << >> ~
6. Unary: ++ -- typeof delete
7. Ternary: ?:
